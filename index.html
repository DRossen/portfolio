Influence Maps

What are Influence Maps?

An Influence Map, in its simplest form, is a grid where each cell holds a value representing some influence in the world. It works by spreading influence from each unit over a set frequency, creating spatial data that describes the unit’s surroundings.The system I’m presenting here takes this concept further by allowing multiple influence maps to be combined at runtime. This enables AI agents to use the combined data to make better strategic and tactical decisions based on their surroundings.For example, let’s say an AI agent wants to throw a grenade but needs to avoid hitting its allies. How can we find a good position to throw it?The solution is simple: we combine enemy(-) and friendly influence(+) within a search radius and retrieve the cell with the lowest influence value. This cell's position describes a location that’s most impacted by enemy presence at the given time.
